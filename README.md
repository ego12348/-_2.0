Висновок до лабораторної роботи №2Під час виконання першої частини лабораторної роботи №2 я:Закріпив навички роботи з операторами розгалуження if, else if та else у мові програмування С.Реалізував обчислення складної математичної функції за варіантом №12, що складається з чотирьох різних гілок розрахунку в залежності від значення аргументу $x$.Опанував роботу з математичною бібліотекою <math.h>, використавши функції sqrt() для добування квадратного кореня та pow() для піднесення до степеня.Вдосконалив вміння використовувати типи даних double для забезпечення високої точності обчислень.Навчився структурувати проект на GitHub, розділяючи лабораторні роботи за різними репозиторіями та оформлюючи звіти з результатами тестування для кожної гілки алгоритму.
